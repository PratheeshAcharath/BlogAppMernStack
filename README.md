# mern-stack-blog-app

Blog Application developed using MERN stack technologies.

## Technologies Used:

* HTML.
* CSS.
* Bootstrap.
* React Bootstrap.
* JavaScript.
* ReactJS.
* NodeJS.
* ExpressJS.
* MongoDB.

## Silent Features:

* User login authentication is added.
* Conditional routing is acheived.
* User can't add blogs without login.
